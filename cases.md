# Тест-кейсы для UI-автотестов

## 1. Создание клиента (Add Customer)

### Описание
Проверка функциональности создания нового клиента с автоматической генерацией данных.

### Предусловия
1. Открыта страница [Bank Manager](https://www.globalsqa.com/angularJs-protractor/BankingProject/#/manager).
2. Пользователь находится на вкладке "Bank Manager Login".

### Шаги
1. Нажать на кнопку Add Customer
2. Сгенерировать случайное 10-значное число для Post Code.
3. Разбить число на пары цифр (5 пар).
4. Преобразовать каждую пару в букву английского алфавита согласно правилам:
    - Если число больше 25, то сбрасывать на 0 (пример: 26 → a, 52 → a).
5. Ввести сгенерированное имя в поле "First Name".
6. Ввести произвольную фамилию в поле "Last Name".
7. Ввести сгенерированный Post Code.
8. Нажать "Add Customer".
9. Подтвердить появившийся alert.

### Ожидаемый результат
- Появляется уведомление "Customer added successfully".
- Во кладке Customers добавляется сгенерированный клиент.

### Постусловия
- Перейти во вкладку "Customers" и удалить созданного клиента. (если требуется)

## 2. Сортировка клиентов по имени (First Name)

### Описание
Проверка работы сортировки клиентов по полю "First Name".

### Предусловия
1. Открыта страница [Bank Manager](https://www.globalsqa.com/angularJs-protractor/BankingProject/#/manager).
2. Пользователь находится на вкладке "Bank Manager Login".
3. В списке клиентов присутствуют клиенты с разными именами.

### Шаги
1. Нажать на кнопку Add Customer
2. Нажать на заголовок колонки "First Name".
3. Проверить порядок сортировки (A-Z).
4. Нажать снова на заголовок колонки "First Name".
5. Проверить порядок сортировки (Z-A).

### Ожидаемый результат
- При первом нажатии клиенты сортируются по возрастанию (Z-A).
- При повторном нажатии клиенты сортируются по убыванию (A-Z).

### Постусловия
- Отменить сортировку (если требуется).

## 3. Удаление клиента

### Описание
Проверка удаления клиента, имя которого ближе всего по длине к среднему арифметическому.

### Предусловия
1. Открыта страница [Bank Manager](https://www.globalsqa.com/angularJs-protractor/BankingProject/#/manager).
2. Пользователь находится на вкладке "Bank Manager Login".
3. В списке клиентов присутствуют несколько имен разной длины.

### Шаги
1. Нажать на кнопку Add Customer
2. Получить список всех имен из таблицы.
3. Определить длину каждого имени.
4. Рассчитать среднее арифметическое длин имен.
5. Найти имя, длина которого ближе всего к среднему.
6. Удалить найденного клиента.
7. Проверить, что имя исчезло из списка.

### Ожидаемый результат
- Клиент успешно удален.
- Имя исчезло из списка клиентов.

### Постусловия
- Удалить последующего клиента по убыванию.
